# CLAUDE.md — 초등 방과후·돌봄 한 장 지도

경기도교육청 **2026 방과후·돌봄 운영 길라잡이**를 korea100 스타일로 도해한
정적 HTML 사이트. GitHub Pages로 <https://jayjayhayhg.github.io/dolbom-map/> 에 배포.

## 프로젝트 규칙

### 1. 콘텐츠 원칙 — 원문만, 추정 금지
- 길라잡이 **원문에 있는 내용만** 쓴다.
- 원문에 없거나 학교마다 다른 것은 반드시 **"⚠ 학교현장 확인 필요"** 섹션으로 분리한다.
- 추정으로 빈칸을 채우지 않는다. 모르면 비워 두고 확인 필요로 표시한다.

### 2. 용어
- 상위 개념은 **"방과후·돌봄"** (가운뎃점 `·` 사용).
- **"늘봄"은 직책명에만** 사용: 늘봄전담실장, 늘봄행정실무사, 늘봄전담실.
- 상위 개념·제도를 가리킬 때 "늘봄"을 단독으로 쓰지 않는다.

### 3. 디자인 토큰 — "길라잡이에 형광펜 친 한 장"
| 토큰 | 값 |
|------|-----|
| paper (배경) | `#F7F8F5` |
| ink (본문) | `#22312B` |
| pine (강조) | `#1E4D3C` |
| 형광펜 (hl) | `#FFE45C` |
| 제목 | Hahmlet |
| 본문 | Pretendard |
| 숫자·라벨 | IBM Plex Mono |

- 콘셉트: **"길라잡이에 형광펜 친 한 장"**.
- 새 페이지는 **기존 페이지의 CSS를 그대로 재사용**한다. 새 토큰·폰트를 임의로 추가하지 않는다.

### 4. 페이지 구조
```
index.html      (허브)
02-matchum.html (초1·2 맞춤형 프로그램)
03-cho3.html    (초3 이용권)
04-suikja.html  (수익자 부담)
05-dolbom.html  (돌봄)
```
- 각 페이지에 **"✎ 현장 메모"** 섹션이 있다.
- 현장 메모 내용은 **운영자가 문장을 주면** 다듬어 넣는다.
- 운영자가 주지 않은 현장 메모를 **창작하지 않는다**.

### 5. 배포
- 수정 후 커밋 메시지는 **한국어로 간단히**.
- `main`에 push하면 GitHub Pages가 자동 반영된다.
- **push 전에 변경 파일 목록을 운영자에게 확인받는다.**
